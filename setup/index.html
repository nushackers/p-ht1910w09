What you need

VirtualBox 6

Our customised Arch Linux ISO

If you have run a VM in VirtualBox before, you should be fine.

If not, please attempt the following set-up process.

Select "Expert Mode".

Start the VM. You should reach the following screen after a while:

The VM may take a while at a screen showing "SHA256 validated".

Once there, you can just shutdown the VM for now by typing:

shutdown -h now

If you encounter an error starting the VM, please check that:

  • Intel VT-x/AMD-V is enabled in your PC's firmware settings
  • (For Windows) Hyper-V is not installed

If you are unsure how to do so, please ask during the workshop tomorrow.
